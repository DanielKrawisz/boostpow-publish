BoostPOW Publish widget demo

DEMO 1

Open the widget with default options and no content hash loaded

const response = await boostPublish.open({
    onPayment: (payment, boostJob) => {
        console.log(payment, boostJob);
    }
});

DEMO 2

Simple text content (boost rank defaults to the last 24 hours)

const response = await boostPublish.open({
    content: "2829b4df5152fb867128f0ea2cffdfe3b7134a98b356eb1a1813b68fd3b83519",
    onPayment: (payment, boostJob) => {
        console.log(payment, boostJob);
    }
});

DEMO 6

Markdown content with boost rank from the last 2 weeks, filtered by tag and category, and using a custom slider with TOP 1, 2 and 3 rank markers configuration.

const response = await boostPublish.open({
    content: "a1ef1e13a09b1f58c5ce7e42f7e24c02c529baa4608652b9895d10b25bb30f5a",
    boostRank: {
        hours: 24 * 14,
        tag: '$osg',
        category: 'B'
    },
    slider: {
        sliderStep: 5,
        rankMarkers: [1, 2, 3]
    },
    onPayment: (payment, boostJob) => {
        console.log(payment, boostJob);
    }
});

DEMO 8

Display simple message, hides the content, display fixed tag and category fields

    content: {
        hash: '2829b4df5152fb867128f0ea2cffdfe3b7134a98b356eb1a1813b68fd3b83519',
        show: false
    },
    tag: { value: '$', disabled: true },
    category: { value: 'B', disabled: true },
    onPayment: (payment, boostJob) => {
        console.log(payment, boostJob);
    }
});
